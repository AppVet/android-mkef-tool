Description

The MasterKey-ExtraField (MKEF) Scanner is an AppVet-compatible online tool that scans Android apps for the presence of the Master Key and Extra Field vulnerabilities.

Possible Results

  • LOW: No Master Key or Extra Field vulnerability was detected.

  • HIGH: Master Key or Extra Field vulnerability was detected.

  • ERROR: An internal problem occurred with this tool.

References

For more information about the Master Key vulnerability, see Android Master Key Vulnerability.
