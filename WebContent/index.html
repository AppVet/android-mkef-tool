Android MasterKey-ExtraField Service

Description

The Android MasterKey-ExtraField Service is an AppVet asynchronous service that detects the presence of the Android Master Key and Extra Field vulnerabilities.

Possible Results

  • LOW: No Master Key or Extra Field vulnerability was detected.

  • MODERATE: Master Key or Extra Field vulnerability was detected.

  • HIGH: An internal problem occurred with this tool service.

References

For more information about the Master Key vulnerability, see Android Master Key Vulnerability.

For more information about the Extra Field vulnerability, see Extra Field Vulnerability.

For more information about asynchronous tool services, see the AppVet documentation.
